🚀 Getting Started

Step 1: Start Your Backend Server

Open terminal and run the backend:

cd backend
npm install
npm start

✓ Backend will run on http://localhost:5000

Step 2: Visit the Pages

Open these HTML files in your browser:

  • 📄 landing.html - Information page
  • 📝 signup.html - Create account
  • 🔐 login.html - Sign in
  • 📊 dashboard.html - Main hub
  • 👥 search.html - Find profiles
  • 🎬 media.html - View media

Step 3: Test the Features

  1. Visit Sign Up page and create a test account
  2. Go to Login and enter your credentials
  3. On Dashboard, you'll see welcome message and stats
  4. Visit Browse Profiles to search and send love requests
  5. Check Media Gallery for photos and videos
  6. View individual Profiles with full details

📡 API Communication

All pages use vanilla JavaScript to communicate with your backend at http://localhost:5000

Main API endpoints used:

  • POST /api/auth/signup - Register new user
  • POST /api/auth/login - User login
  • GET /api/users/search - Search profiles
  • GET /api/users/:id - Get profile details
  • POST /api/match/love-request - Send love request
  • GET /api/media - Fetch media gallery

❓ Quick Troubleshooting

Can't login? Make sure backend server is running on localhost:5000

Pages won't load data? Check if API is accessible and token is valid

Sign up fails? Remember: Uganda=Female, Europe=Male, must be 18+

Mobile view broken? All pages are fully responsive, try different screen size
